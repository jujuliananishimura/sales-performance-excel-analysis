# Sales Performance Analysis (Excel)

## Project Overview
This project analyzes sales data using Excel pivot tables to identify product performance, sales trends, and the impact of discounts.

## Tools Used
- Excel (Pivot Tables, Charts, Calculated Fields)

## Key Insights
- Motorcycles generate higher total sales than Classic Cars
- Sales show monthly seasonality
- Higher discounts are associated with higher sales volume
- Discount strategy can be optimized to improve profitability

## Skills Demonstrated
- Data cleaning
- Data analysis
- Pivot tables
- Business insights
